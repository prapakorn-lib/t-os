**‡∏ó‡∏ö‡∏ó‡∏ß‡∏ô‡∏Ñ‡∏ß‡∏≤‡∏°‡∏£‡∏π‡πâ:**
- **r (read) = 4**
- **w (write) = 2**
- **x (execute) = 1**
- **rwx = 4+2+1 = 7**
- **rw- = 4+2 = 6**
- **r-- = 4**

**6.1 chmod ‡∏î‡πâ‡∏ß‡∏¢‡∏ï‡∏±‡∏ß‡πÄ‡∏•‡∏Ç (Numeric Mode):**
```bash

# 644 = rw-r--r-- (‡πÑ‡∏ü‡∏•‡πå‡∏õ‡∏Å‡∏ï‡∏¥)

# 755 = rwxr-xr-x (script ‡∏ó‡∏µ‡πà‡∏£‡∏±‡∏ô‡πÑ‡∏î‡πâ)
chmod 755 test_script.sh

# 700 = rwx------ (private script)
chmod 700 private_script.sh

# 777 = rwxrwxrwx (everyone can do everything - ‡∏≠‡∏±‡∏ô‡∏ï‡∏£‡∏≤‡∏¢!)
chmod 777 public_file.txt

# ‡∏ï‡∏£‡∏ß‡∏à‡∏™‡∏≠‡∏ö

```

**‡∏ï‡∏±‡∏ß‡πÄ‡∏•‡∏Ç‡∏ó‡∏µ‡πà‡πÉ‡∏ä‡πâ‡∏ö‡πà‡∏≠‡∏¢:**
- **644:** ‡πÑ‡∏ü‡∏•‡πå‡∏ó‡∏±‡πà‡∏ß‡πÑ‡∏õ (owner: rw, others: r)
- **664:** ‡πÑ‡∏ü‡∏•‡πå‡∏ó‡∏µ‡πà group ‡πÅ‡∏Å‡πâ‡πÑ‡∏Ç‡πÑ‡∏î‡πâ
- **755:** executable files ‡πÅ‡∏•‡∏∞ directories
- **700:** private executable
- **600:** private file (owner only)
- **777:** public everything (‡πÑ‡∏°‡πà‡πÅ‡∏ô‡∏∞‡∏ô‡∏≥!)

**6.2 chmod ‡∏î‡πâ‡∏ß‡∏¢ Symbolic Mode:**
```bash

chmod u+x test_script.sh

# ‡∏•‡∏ö‡∏™‡∏¥‡∏ó‡∏ò‡∏¥‡πå write ‡∏à‡∏≤‡∏Å group

# ‡∏ï‡∏±‡πâ‡∏á‡∏™‡∏¥‡∏ó‡∏ò‡∏¥‡πå‡πÅ‡∏ô‡πà‡∏ô‡∏≠‡∏ô

chmod +x script.sh

# ‡∏•‡∏ö write ‡∏à‡∏≤‡∏Å‡∏ó‡∏∏‡∏Å‡∏Ñ‡∏ô

```

**‡∏™‡∏±‡∏ç‡∏•‡∏±‡∏Å‡∏©‡∏ì‡πå:**
- **u:** user (owner)
- **g:** group
- **o:** others
- **a:** all (everyone)
